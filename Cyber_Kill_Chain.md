# Cyber Kill Chain — Notes

1. Reconnaissance — attacker gathers information
2. Weaponization — creates malware or exploit
3. Delivery — sends malware (email, USB, website)
4. Exploitation — vulnerability exploited
5. Installation — malware installed
6. C2 (Command and Control) — remote communication
7. Actions — data theft, ransomware, lateral movement

### SOC Usage
- Helps determine attacker stage.
- Helps predict next steps.
- Useful for timeline creation.
- Supports IR decision-making.
